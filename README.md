* #25 Alfrescoにワークフローを追加するデモ実装
* #29 Alfresco上のアクティビティモニタリング（elasticsearchへのpush）を追加
* #38 ユーザのhomeスペースをWebDAVの固定名フォルダに割り当てるデモ実装
* #40 QRコードを読み取って文書を更新するデモ実装
* #42 Alfresco JS-APIを使ったコンテンツモデルデザイナのデモ実装
